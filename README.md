# hypermodern_python

Redesigned course based on older articles "Hypermodern Python" by Claudio Jolowicz.

Provides a pywc utility as an example project, which is similar to wc.

## Installation and usage

1. [Install](https://docs.astral.sh/uv/getting-started/installation/#standalone-installer) `uv`.
1. Run `uv run pywc --help` for command line interface.
1. Run `uv run poe doc_host` to create localhost documentation on [localhost](http://localhost:8000/index.html).
