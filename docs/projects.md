# Project Topics

Students should select one topic from the list below. Topics must be unique per student (no repetitions) while possible.\
You may start working on the project immediately. There will be no significant assistance with the core logic.

**Recommendation:** Avoid implementing file processing at this early stage, before installing `ruff`, unless you know about `pathlib`.

1. **Image Filter Application**\
   For each image in a folder, apply the specified filter and save the result.\
   Available filters: blur (Gaussian blur) and sharpening.

1. **Color Palette Analyzer**\
   For each image in a folder, extract the dominant colors and display the results.\
   Generate histograms for each color channel (mean, median, standard deviation), determine the dominant color, calculate brightness level, and evaluate saturation.

1. **Edge and Contour Detection**\
   For each image in a folder, detect edges and contours.\
   Key techniques: Sobel operator, Laplacian operator.

1. **Image Resizing with Aspect Ratio Options**\
   For each image in a folder, resize images according to user-specified dimensions while maintaining or ignoring the aspect ratio (with stretching/compression options).

1. **Watermark Overlay**\
   For each image in a folder, overlay a specified watermark on the image.

1. **Image Quality Assessment**\
   Sort input images by their quality.\
   Key metrics: Laplacian variance, sharpness/blur metrics.

1. **Grayscale Image Conversion**\
   For each image in a folder, convert the image to grayscale using the algorithm selected by the user.

1. **Custom Topic**\
   If you have your own idea for a project, feel free to propose it in person or via message.
